the application grouped by objects.

=== Adding Recent Items to the Menu

To add the *Recent Items* module to the link:ios/app-menu[app menu]:

[width="100%",cols="50%,50%",]
|===
|In CT Mobile Control Panel a|
. Go to link:ios/ct-mobile-control-panel-app-menu[CT Mobile Control
Panel: App Menu] tab.
. {blank}
. Click the
image:62562609.png[]
button to open the *Add menu item* tool.
. Select the *Recent Items* item.
. Click *Add* to close the *Add menu item* tool.
. Click *Save*.

|In CT Mobile Control Panel 2.0 a|
. Go to link:ios/ct-mobile-control-panel-app-menu-new[CT Mobile Control
Panel 2.0: App Menu] tab.
. Select the required profile in the *Profile* picklist. To apply
settings to all the profiles, select *General Settings*.
. Click *Show Module Menu*.
. On the *All* or *Application Modules* tab, select the **Recent
Items** item.
. Click *Save*.

|===

The setup is complete.
