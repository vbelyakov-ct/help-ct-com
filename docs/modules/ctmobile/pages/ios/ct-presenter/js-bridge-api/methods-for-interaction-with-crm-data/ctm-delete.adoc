= ctm.delete

The method of the xref:ios/ct-presenter/js-bridge-api/index.adoc[JS Bridge] functionality is
intended to bulk delete fields of records of offline objects:

* xref:ios/getting-started/application-permission-settings.adoc[the user profile] should
have permission to delete records of the required object.
* xref:ios/admin-guide/managing-offline-objects/index.adoc#h2_1534686659[permission to delete]
the required offline object is allowed.

[[h2_905713055]]
=== Query Format



[.apiobject]#SObject# is the JS object that indicates the API
name of the offline object and the ID of the record to delete.
