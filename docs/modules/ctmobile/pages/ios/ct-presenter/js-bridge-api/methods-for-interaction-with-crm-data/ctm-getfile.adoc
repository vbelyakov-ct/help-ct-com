= ctm.getFile

The method of xref:ios/ct-presenter/js-bridge-api/index.adoc[JS Bridge] functionality is intended to obtain a local path to the file attachment of the [.object]#Attachment# or [.object]#ContentVersion# objects by using their ID.

[[h2_905713055]]
== Query format

[source,javascript]
----
ctm.getFile(Id)
----

[.apiobject]#Id# is the ID of the _Attachment_ record that should start from [.apiobject]#00P# or the ID of the _ContentVersion_ record that should start with [.apiobject]#068#.

[[h2_442663712]]
== Handler
