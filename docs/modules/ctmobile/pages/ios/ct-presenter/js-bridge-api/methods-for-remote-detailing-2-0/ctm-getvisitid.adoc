= ctm.getVisitId

intended to return the Id of the *Activity* record, from which the
*Remote Detailing* or *Self Detailing* meeting was launched.

[[h2__905745855]]
=== Query Format



[.apiobject]#visitId# is the Id of the corresponding *Activity*
record.
