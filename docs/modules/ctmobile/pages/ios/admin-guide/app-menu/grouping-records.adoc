= Grouping Records

Initially, the records are grouped in alphabetical order. You can rearrange the records according to the selected field.

To group the records:

[tabs]
====
In CT Mobile Control Panel::
+
--
. Go to xref:ios/admin-guide/ct-mobile-control-panel/ct-mobile-control-panel-app-menu.adoc[CT Mobile Control Panel: App Menu].
. {blank}
include::partial$settings-location.adoc[]
. Select an object from the *Selected* list.
. In the *Group records by* picklist, select a field for grouping.
. Select ascending/descending records grouping order.
. Click *Save*.
+
image:47752559.png[]
--
In CT Mobile Control Panel 2.0::
+
--
. Go to the xref:ios/admin-guide/ct-mobile-control-panel-new/ct-mobile-control-panel-app-menu-new.adoc[CT Mobile Control Panel 2.0: App Menu].
. {blank}
include::partial$settings-profile.adoc[]
. Select an object from the *Selected* list.
. In the *Group records by* picklist, select a field for grouping.
. Select ascending/descending records grouping order.
. Click *Save*.
+
image:ctm-control-panel-new-app-menu-grouping-records.png[]
--
====

xref:ios/mobile-application/synchronization/synchronization-launch/index.adoc#h3_1369866827[Launch the full synchronization] on the mobile device to apply changes. Records are grouped according to the specified field.
