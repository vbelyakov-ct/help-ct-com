= Creating CLM Presentation

CLM presentations can be created slide by slide in Salesforce.

* xref:android/creating-a-slide[Creating a Slide]
* xref:android/creating-a-template[Creating a Template]
* xref:android/creating-a-placeholder[Creating a Placeholder]
* xref:android/creating-a-widget[Creating a Widget]



On the *Application Editor* tab, select the CLM presentation in the
picklist.



In the *CLM-presentation* section:

* Click *New* to create a new CLM presentation.
* Click *Clone* to copy the selected CLM presentation to a new CLM
presentation record.
* Click *Delete* to remove the CLM presentation.
* Click *Upload PPT file* to create a CLM presentation based on the
xref:android/knowledge-base/ct-presenter/application-editor/creating-clm-presentation-from-powerpoint[PowerPoint],
xref:android/knowledge-base/ct-presenter/application-editor/creating-clm-presentation-from-pdf[PDF], or
xref:android/knowledge-base/ct-presenter/application-editor/creating-clm-presentation-from-jpeg-jpg[JPEG/JPG] file.
* Click xref:android/publishing-clm-presentations[Publish] to apply all
settings for the CLM presentation.

image:application_editor_1.png[]



In the *Slides* section:

* Click *Add new* to add a new slide.
* Click *Save* to save changes.
* Click *Cancel* to roll back all unsaved changes.
* Click *Delete* next to a slide to remove a slide.
* Click the xref:android/creating-a-slide[Slide] name to edit a slide.
* Click the xref:android/creating-a-template[Template] name to edit a
template.
* Specify whether the slide is active and can be used in CLM
presentation or not.
* Specify the *Required* attribute and a category, if needed.
* Specify slides in
xref:android/clm-navigation-in-clm-presentations[branches].
