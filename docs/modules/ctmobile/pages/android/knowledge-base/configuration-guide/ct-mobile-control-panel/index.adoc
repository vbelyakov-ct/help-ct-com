= CT Mobile Control Panel

*CT Mobile Control Panel* is a Salesforce interface for connecting the CT Mobile solution to the customer's target system and for specifying  all the necessary settings to personalize the CT Mobile app.

WARNING: A user configuring a CT Mobile solution must have a separate profile and an assigned xref:android/quick-reference-guides/application-permission-settings.adoc[CT Mobile Administrator] permission set.

* All customizations for the selected profile will be saved in xref:android/knowledge-base/configuration-guide/custom-settings/index.adoc[custom settings] or as xref:android/knowledge-base/configuration-guide/ctm-settings/index.adoc[CTM Settings] records.
* Settings for xref:android/knowledge-base/configuration-guide/ctm-settings/ctm-settings-menu.adoc[menu], xref:android/knowledge-base/configuration-guide/ctm-settings/ctm-settings-offline-objects.adoc[offline objects], and xref:android/knowledge-base/configuration-guide/ctm-settings/ctm-settings-standard-related-list.adoc[standard related lists] on the *CT Mobile Control Panel* are displayed based on the *CTM Settings* records. Other settings are displayed according to the records of the custom settings.
