= Sign Credentials

_Sign Credentials_ is a protected custom setting that is responsible xref:admin-guide/connect-salesforce-with-the-application-service-and-e-signature-provider.adoc#h2__1758912760[for connecting] the Salesforce organization with the CT Sign package and the Application Service.

'''''

API name in Salesforce: [.apiobject]#ctsign\__SignCredentials__c#

'''''

[cols="15%,20%,10%,55%"]
|===
|*Field* |*API Name* |*Type* |*Description*

|Token |[.apiobject]#ctsign\__Token__c# |Text(255) |The Application Service access token.

|URL |[.apiobject]#ctsign\__URL__c# |Text(255) |The URL address to authorize in the Application Service: `https://sign.customertimes.com`
|===
