= 6. Configuring Application Settings

To manage the _Global_ record of *Application Settings*, or create a new  one if you want to specify more than one marketing cycle:

. Go to *Setup→ Custom Code → Custom Settings →* click *Manage* next to *Application Settings*.
. Click *Edit* next to the _Global_ record, or click *New* to create a new record.
. Specify the required parameters.
* Read the description of each parameter xref:admin-guide/cpg-custom-settings/application-settings.adoc[here].
* It is necessary to update records of *Application Settings* in case of using xref:admin-guide/targeting-and-marketing-cycles-management/add-a-new-division.adoc[new divisions].
. Click *Save*.

The setup is complete.
