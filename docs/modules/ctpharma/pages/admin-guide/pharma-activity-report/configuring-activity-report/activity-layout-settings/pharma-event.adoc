The *Pharma Event* field set is related to the _Pharma Event_ record type. Now, you can customize the *Pharma Event* field set.

To define which fields will be available on the Pharma Activity Report for _Pharma Event_ record type:

. Go to *Setup*(1) → *Object Manager*(2) → *Pharma Activity*(3) → *Field Sets*(4).
+
The list of the field sets opens.
. Click *Pharma Event *(5).
+
image:58829331.png[]
+
The Field Set editor opens.
. Edit a visible field set by dragging available fields to the *In the Field Set* list and click *Save*(6).
+
image:58829332.png[]

The Pharma Event field set is customized.

According to the settings, the main section is displayed on the *Pharma Activity Report* page.
